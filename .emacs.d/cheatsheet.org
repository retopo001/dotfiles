#+TITLE: Emacs Cheatsheet


* Toggles (C-c t prefix)
| Key       | Action              |
|-----------+---------------------|
| C-c t w   | Toggle word wrap    |

* Navigation (Custom C-c bindings)
| Key       | Action                |
|-----------+-----------------------|
| C-c i     | Jump to function      |
| C-c l     | Search buffer         |
| C-c r     | Search project (rg)   |
| C-c f     | Find file in project  |
| C-c g     | Magit (git)           |
| C-c d     | Devdocs lookup        |
| C-c w     | EWW (web browser)     |
| C-c ?     | This cheatsheet       |
| C-c p     | Palette (notes)       |

* Files
| Key       | Action         |
|-----------+----------------|
| C-x C-f   | Open file      |
| C-x C-s   | Save           |
| C-x C-w   | Save as        |
| C-x k     | Kill buffer    |

* Buffers
| Key       | Action              |
|-----------+---------------------|
| C-x b     | Switch buffer       |
| C-x C-b   | List buffers        |
| C-x C-B   | ibuffer (powerful)  |

* Windows
| Key       | Action              |
|-----------+---------------------|
| C-x 2     | Split horizontal    |
| C-x 3     | Split vertical      |
| C-x 1     | Close other windows |
| C-x 0     | Close this window   |
| C-x o     | Other window        |

* Movement
| Key       | Action              |
|-----------+---------------------|
| C-n / C-p | Next/prev line      |
| C-f / C-b | Forward/back char   |
| M-f / M-b | Forward/back word   |
| C-a / C-e | Start/end of line   |
| M-< / M-> | Start/end of buffer |
| M-g g     | Go to line          |

* Search
| Key       | Action              |
|-----------+---------------------|
| C-s       | Search forward      |
| C-r       | Search backward     |
| M-.       | Go to definition    |
| M-?       | Find references     |

* Editing
| Key       | Action              |
|-----------+---------------------|
| C-SPC     | Set mark            |
| C-w       | Kill region (cut)   |
| M-w       | Copy region         |
| C-y       | Yank (paste)        |
| C-/       | Undo                |
| M-;       | Comment/uncomment   |

* Help
| Key       | Action              |
|-----------+---------------------|
| C-h k     | Describe key        |
| C-h f     | Describe function   |
| C-h v     | Describe variable   |
| C-h m     | Describe mode       |
| C-h b     | All bindings        |

* Stamp Macros (HYPER key on Planck EZ)
HYPER = hold Alt+Shift+Ctrl+Cmd (Oryx config)
| Key             | Action                            |
|-----------------+-----------------------------------|
| Hyper+t         | Insert "TODO: "                   |
| Hyper+s         | Insert "SHOULD BE: "              |
| Hyper+a         | Insert timestamp                  |
| Hyper+Backspace | Delete timestamp at point         |
| Hyper+Delete    | Delete timestamp at point         |
| Hyper+End       | Update timestamp to current time  |
| Hyper+Home      | Cycle through timestamp history   |

Timestamp format: 26-01-18_1845.30.Sun
Cycle wraps around - keeps cycling through all previous values.

* Calendar (C-c c prefix)
| Key       | Action              |
|-----------+---------------------|
| C-c c c   | Show khal calendar  |
| C-c c a   | Show agenda (30d)   |
| C-c c s   | Sync via vdirsyncer |

* which-key
Press any prefix (like C-x or C-c) and wait - shows available completions.
- C-h while showing: more options/pages
